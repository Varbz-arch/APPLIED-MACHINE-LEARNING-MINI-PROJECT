# 🌍 GEOHEALTH-AI

## 🩺 Overview

GEOHEALTH-AI is a Machine Learning-based healthcare prediction system that analyzes health-related data across different states of India. It predicts disease outcomes using historical data, seasonal information, and geographical features.

The project combines data preprocessing, machine learning, and a Flask web application to provide an easy-to-use interface for disease prediction.

---

## ✨ Features

- 🧠 Machine Learning disease prediction
- 📍 State-wise health analysis
- 📅 Seasonal and monthly disease prediction
- 🌐 Interactive web interface using Flask
- 📊 Feature importance visualization
- 📈 Confusion matrix evaluation
- 🔄 Model retraining scripts

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/GEOHEALTH-AI.git
```

Move into the project folder

```bash
cd GEOHEALTH-AI
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
python app.py
```
